# C Programming Projects

This repository contains beginner C programming projects completed while studying Computer Science at the University of Nebraska Omaha.

## Projects

### Integer Range Generator
Program asks the user for an integer and prints the sequence of numbers up to ten numbers after it.

Concepts used:
- while loops
- user input
- control flow

### Time Conversion Calculator
Program converts minutes into hours and remaining minutes.

Concepts used:
- division and modulus
- macros
- loops
